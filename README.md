
## App Quickstart

1. `npm install -g react-native-cli`
2. clone shit-vcs-say, `cd ./shit-vcs-say/app`
4. `npm install`
5. `react-native run-ios`

## Server Quickstart

1. `cd ./shit-vcs-say/server`
2. `npm install`
3. `touch ./models/config.json` (database config, ask for credentials)
4. `npm start`
